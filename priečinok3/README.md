Open index_final.html in your browser to try the Vue 3 example.

How to run:
- Double-click `index_final.html` in File Explorer, or
- Right-click and choose "Open with" and pick your browser.

Notes:
- This file uses the Vue 3 ESM browser build via CDN, so no local build step is required.
- If you edit the file, refresh the browser to see changes.

Quick checks:
- Add an expense using the inputs and "Add Item".
- Remove items with the ❌ button.
- Toggle inline editing with the ✏️ button.
